# Vue3-admin-manage

## 简介

vue3-admin-manage 是一个基于vue3开发的后台管理模版。集合了`webpack4`,`TypeScript`,`Element-Plus`等主流技术，提供多种组件示例，旨在快速搭建后台管理系统。

## 在线预览

- [vue3-admin-manage](http://www.lubaojun.com/)

## 使用说明

```
- node版本 >= v12.0
- vue-cli >= 4.5.0
- 支持现代浏览器以及Chrome >= 51
```

## 运行项目

安装依赖

```
npm install
```

启动本地调试

```
npm run serve:dev
```

编译打包，生成编译后的目录

```
npm run build:dev
```
